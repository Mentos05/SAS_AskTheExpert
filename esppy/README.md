<br>
SAS Event Stream Processing is perfect if you want to analyze streaming data, uncover hidden insights with AI and make real-time, intelligent decisions.<br>
In this example I demonstrate how to use a trained object detection model to anaylze video footage from an airport in real-time.<br>
